# 2026 Q1 涨价前"心理缓冲层"设计

## 🎯 目标

在不伤害 Sora 价值感的前提下，为 Q1 涨价做准备。

## 📋 三层心理缓冲策略

### Layer 1（现在立刻）：命名与定位（不涨价）

**目标**：让用户接受"分层"概念，而不是"贵/便宜"

#### Sora 定位
- **名称**：Sora (Fast Iteration)
- **描述**：Everyday creation / fast iteration（日常高频）
- **页面上只写**：use-case & output quality difference
- **绝不写**："cheap / low / basic"

#### Veo Pro 定位
- **名称**：Veo Pro (Premium Final Render)
- **描述**：Premium finish / higher fidelity（高级成片）
- **页面上只写**：when to use（使用时机）
- **绝不写**："better than Sora"

#### UI 文案示例

```
Sora Preview
Fast, lightweight video generation for early exploration.

Veo Pro
Preferred when final quality and sound matter.
```

### Layer 2（Q1 中段）：把 4.9 从"低价"改成"权限型"

**目标**：让用户认为"我买的是试用权限，不是廉价积分包"

#### 新定位：Starter Access (7 days)

**页面文案**：
```
Starter Access (7 days) includes:

• Sora: 10 credits available now
• Bonus credits: time-limited (expires in 7 days)
• Daily rate limit + device binding
```

#### 核心变化

| 旧方案 | 新方案 |
|--------|--------|
| $4.9 = 永久积分 | $4.9 = 7 天访问权限 |
| "买积分" | "买权限" |
| 可囤积 | 不可囤积（自动过期） |
| 价值感低 | 价值感高（试用期概念） |

#### 用户心理变化

- ✅ "我买的是试用权限，不是廉价积分包"
- ✅ 价值感立刻上来
- ✅ 薅羊毛也难了（时间限制）

### Layer 3（Q1 后段）：涨价不改入门门槛，只改"升级路径"

**目标**：让用户感知"Pro 更值"，不是"你在变贵"

#### 涨价策略

**不动**：
- ❌ 4.9 的数字（它是广告入口）
- ❌ Starter Access 的基本概念

**动**：
- ✅ Starter 的"可用次数/上限"
- ✅ Veo Pro 的 "bundle / pack" 结构
- ✅ 引入 "Priority queue / faster processing" 作为 Pro 的明确权益

#### 具体实施

1. **Starter Access 调整**
   ```
   旧：15 次 Sora / 7 天
   新：12 次 Sora / 7 天（减少 20%，但强调"更专注的试用"）
   ```

2. **Veo Pro Bundle 引入**
   ```
   新选项：Veo Pro Pack
   - 5 次 Veo Pro: $18.99（单次 $3.80，比 $3.99 便宜）
   - 10 次 Veo Pro: $34.99（单次 $3.50）
   - 20 次 Veo Pro: $64.99（单次 $3.25）
   ```

3. **Pro 权益明确化**
   ```
   Veo Pro 权益：
   • Priority queue（优先队列）
   • Faster processing（更快处理）
   • Higher success rate（更高成功率）
   • Auto-refund on failure（失败自动返还）
   ```

#### 用户感知

- ✅ "Pro 更值"（Bundle 折扣）
- ✅ "不是你在变贵"（入门门槛不变）
- ✅ "我升级是因为需要这些权益"（不是被迫）

## 📊 实施时间表

## 🎯 验收指标

### Layer 1 验收
- ✅ 用户不再问"Sora 是不是便宜版"
- ✅ 用户理解"Sora = 迭代，Veo = 最终"

### Layer 2 验收
- ✅ Starter Access 购买率 ≥ 15%
- ✅ 用户反馈"试用权限"概念清晰
- ✅ 薅羊毛行为减少 50%+

### Layer 3 验收
- ✅ Veo Pro Bundle 购买率 ≥ 30%
- ✅ 用户留存率提升
- ✅ 收入增长 ≥ 20%

## 💡 关键洞察

1. **先改"权益表达"，再动"价格数字"**
2. **4.9 是广告入口，不能动**
3. **涨价不改入门门槛，只改升级路径**
4. **用户感知是"Pro 更值"，不是"你在变贵"**
