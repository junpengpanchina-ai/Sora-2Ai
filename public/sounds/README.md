# 背景音乐文件说明

## 圣诞节背景音乐

请将圣诞节背景音乐文件放置在此目录下，文件名为：`christmas-bgm.mp3`

### 推荐音乐类型：
- 圣诞颂歌（Christmas Carols）
- 轻快的圣诞器乐（Christmas Instrumental）
- 环境圣诞音乐（Ambient Christmas Music）

### 音乐要求：
- 格式：MP3
- 时长：建议 2-5 分钟（会自动循环播放）
- 音量：建议适中，代码中已设置为 30% 音量
- 内容：无版权或已获得使用许可的圣诞音乐

### 如何添加音乐：
1. 将音乐文件重命名为 `christmas-bgm.mp3`
2. 放置在 `public/sounds/` 目录下
3. 确保文件可以通过 `/sounds/christmas-bgm.mp3` 访问

### 免费圣诞音乐资源：
- [Freesound.org](https://freesound.org) - 搜索 "christmas music"
- [Pixabay Music](https://pixabay.com/music/) - 免费无版权音乐
- [Zapsplat](https://www.zapsplat.com) - 免费音效库

### 注意事项：
- 某些浏览器可能会阻止自动播放音频，需要用户先与页面交互
- 音乐会在切换到圣诞节主题时自动播放，切换到其他主题时自动停止
