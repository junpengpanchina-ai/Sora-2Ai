# 外链快速检查清单

## ⚡ 5分钟快速检查

### ✅ 步骤 1: Google Search Console（2分钟）

1. **访问**: https://search.google.com/search-console
2. **选择你的网站属性**
3. **点击"链接"报告**

**记录以下数据**：
```
外部链接总数: _____
引用域名数: _____
最常链接的页面: _____
```

**判断**：
- ✅ **足够**: 外部链接 > 50 且 引用域名 > 20
- ⚠️ **一般**: 外部链接 20-50 或 引用域名 10-20
- ❌ **不足**: 外部链接 < 20 或 引用域名 < 10

### ✅ 步骤 2: 检查抓取统计（1分钟）

1. **在 Search Console 中**: 点击 **设置** > **抓取统计信息**

**记录以下数据**：
```
每天抓取的网页数: _____
每天下载的千字节数: _____
下载网页所需的时间: _____秒
```

**判断**：
- ✅ **足够**: 每天抓取 > 100页，时间 < 1秒
- ⚠️ **一般**: 每天抓取 10-100页
- ❌ **不足**: 每天抓取 < 10页 或 时间 > 3秒

### ✅ 步骤 3: Google 搜索检查（1分钟）

在 Google 搜索框中分别输入：

```
1. link:your-domain.com
   结果数量: _____
   
2. "your-domain.com"
   结果数量: _____
   
3. site:your-domain.com
   索引页面数: _____
```

**判断**：
- ✅ **足够**: `link:` 结果 > 20，`site:` 索引页面 > 你的页面总数 × 0.8
- ⚠️ **一般**: `link:` 结果 5-20，索引率 50-80%
- ❌ **不足**: `link:` 结果 < 5 或 索引率 < 50%

### ✅ 步骤 4: 检查页面索引状态（1分钟）

1. **在 Search Console 中使用 URL 检查工具**
2. **输入几个重要的长尾词页面URL**
3. **检查每个页面**：

```
页面 1: _____
  - 是否可编入索引: ✅/❌
  - 上次抓取时间: _____
  
页面 2: _____
  - 是否可编入索引: ✅/❌
  - 上次抓取时间: _____
  
页面 3: _____
  - 是否可编入索引: ✅/❌
  - 上次抓取时间: _____
```

**判断**：
- ✅ **足够**: 所有重要页面都已索引，抓取时间 < 7天
- ⚠️ **一般**: 大部分页面已索引，抓取时间 7-30天
- ❌ **不足**: 很多页面未索引，或抓取时间 > 30天

## 📊 综合评估

### 计算总分

根据以上检查，给每个步骤打分：
- ✅ 足够 = 3分
- ⚠️ 一般 = 2分
- ❌ 不足 = 1分

**总分**: _____ / 12分

### 评估结果

- **9-12分**: 🟢 **外链充足** - 继续维护和增加高质量外链
- **6-8分**: 🟡 **外链一般** - 需要增加外链建设
- **<6分**: 🔴 **外链不足** - 急需外链建设计划
