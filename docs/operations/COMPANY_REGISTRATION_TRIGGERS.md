# 公司注册触发条件表

> **用途**：理性决策何时注册公司
> **原则**：不被焦虑推着走，按条件触发

---

## 核心判断

```
注册公司不是"升级"，是"责任切换"。

在你没被迫承担责任前，
保持个人身份 = 最低风险 + 最大灵活性。
```

---

## 🚦 必须注册公司（满足任一即触发）

| 触发条件 | 标准 | 原因 |
|----------|------|------|
| **Enterprise 客户要求** | 客户明确要求签合同 | 法律主体要求 |
| **年收入** | ≥ USD $50,000 | 税务合规 + 责任隔离 |
| **单客户金额** | ≥ USD $10,000 一次性 | 风险集中 |
| **长期服务承诺** | 客户要求 SLA / 年付合同 | 需要法律主体 |
| **平台要求** | Stripe / 银行要求公司主体 | 合规要求 |
| **风险敞口** | 客户要求明确责任承担 | 责任隔离 |
| **雇佣员工** | 任何形式的雇佣关系 | 劳动法要求 |

**规则**：满足 1 条 → 立即启动注册

---

## 🟡 强烈建议注册（可缓冲 30-60 天）

| 情况 | 说明 | 缓冲建议 |
|------|------|----------|
| **同时 2+ 企业客户** | 风险分散需要 | 30 天内注册 |
| **外包/承包商** | 责任扩散 | 60 天内注册 |
| **投资人正式沟通** | 尽调需要 | 沟通前完成 |
| **跨境业务** | 税务合规 | 按目标市场要求 |
| **年收入 $30k-$50k** | 接近阈值 | 提前准备 |

---

## 🟢 不需要注册（安心区）

| 情况 | 状态 | 说明 |
|------|------|------|
| 个人用户为主 | ✅ OK | 标准 B2C 模式 |
| 一次性充值 | ✅ OK | 无长期承诺 |
| 无合同要求 | ✅ OK | ToS 足够 |
| 年收入 < $30k | ✅ OK | 个人所得税即可 |
| Credits 模式 | ✅ OK | 预付无承诺 |
| Preview/Beta | ✅ OK | 明确实验性质 |

## ⚠️ 心理误判提醒（v1.1 新增）

```
❗ "觉得自己应该注册公司" ≠ 触发条件

以下情况都不是注册公司的理由：
- "感觉更专业"
- "别人都有公司"
- "万一以后需要"
- "看起来更正规"
- "有点焦虑"

这些是情绪，不是条件。

真正的触发条件只有上面列出的那些。
在条件没有满足之前，保持个人身份是最优策略。
```

### 自检问题

当你想注册公司时，问自己：

1. **有客户明确要求合同吗？** → 没有 = 不需要
2. **年收入超过 $50k 了吗？** → 没有 = 不需要
3. **单客户超过 $10k 了吗？** → 没有 = 不需要
4. **Stripe/银行要求了吗？** → 没有 = 不需要
5. **要雇人了吗？** → 没有 = 不需要

如果 5 个都是"没有"，那就是**情绪驱动**，不是**条件驱动**。

## 常见问题

### Q: 没有公司能收 Enterprise 客户吗？

```
A: 可以，但有限制：
   - 只能用 Credits 模式
   - 不能签正式合同
   - 不能承诺 SLA
   - 客户需要接受 ToS
```

### Q: 什么时候"必须"有公司？

```
A: 三种情况：
   1. 客户采购流程要求
   2. 平台（Stripe/银行）要求
   3. 税务合规要求
```

### Q: 注册公司会增加多少成本？

```
A: 取决于地区：
   - 美国 LLC: $300-500/年
   - 新加坡: $2,000/年
   - 香港: $1,500/年
   
   加上会计、报税等，预算 $2,000-5,000/年
```

### Q: 可以先收钱后注册吗？

```
A: 可以，只要：
   - 用个人身份合法收款
   - 正确申报个人所得税
   - ToS 明确个人身份
   - 客户接受 Credits 模式
```
