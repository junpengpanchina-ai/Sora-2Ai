# 14 天极简观测表

> **版本**: v1.0  
> **生效日期**: 2026-01-24  
> **目标**: 用最少指标，判断「Google 是在犹豫，还是在拒绝」

---

## 每天只看 5 个指标

| # | 指标 | 来源 | 你要看的是 |
|---|------|------|------------|
| 1 | **Discovered URLs** | GSC → Pages | 不是数量，是增长 |
| 2 | **Indexed URLs** | GSC → Pages | 不是数量，是比例 |
| 3 | **Index Rate** | 你的 Dashboard | 是否跨阈值 |
| 4 | **Crawled – not indexed** | GSC → Pages | 是否"堆积" |
| 5 | **Crawl Stats** | GSC → Settings | 是否下降 |

---

## 每天只问 3 个问题

```
1. Index Rate 是上升 / 横盘 / 下降？
2. "Crawled – not indexed" 是在消化，还是在堆积？
3. Google 还在频繁来吗？（Crawl Stats）
```

**规则**：如果你回答不了其中任意一个 = 不允许扩容

## 各阶段指导

### Day 1–3：不下结论

```
✅ 做：
- 每天记录 5 个指标
- 确认数据来源正确
- 建立基线

❌ 不做：
- 不下任何结论
- 不做任何调整
- 不惊慌
```

### Day 4–7：看趋势，不动作

```
✅ 做：
- 观察趋势方向
- 识别模式
- 记录异常

❌ 不做：
- 不改 sitemap
- 不加 core
- 不扩容
```

### Day 8–10：初步判断区间

```
✅ 做：
- 计算 7 日平均
- 判断 Zone（🟢/🟡/🔴）
- 准备决策依据

❌ 不做：
- 仍然不执行决策
- 只是准备
```

### Day 14：决策日

```
✅ 允许做：
- 根据 Index Rate 阈值做出决策
- 如果 🟢：可考虑下一步
- 如果 🟡：继续观察
- 如果 🔴：进入诊断模式

📋 必须记录：
- 最终 Index Rate
- 决策理由
- 下一步计划
```

## 异常信号识别

### ⚠️ 警告信号

| 信号 | 含义 | 动作 |
|------|------|------|
| Discovered 不增长 | sitemap 可能未被消费 | 检查 sitemap 提交状态 |
| Indexed 停滞 | Google 在犹豫 | 等待，不干预 |
| Index Rate 持续下降 | 内容/模板问题 | 准备诊断 |

### 🚨 危险信号

| 信号 | 含义 | 动作 |
|------|------|------|
| Crawled-not-indexed 激增 | Google 明确拒绝 | 立即进入 RED |
| Crawl Stats 断崖下降 | 可能被降权 | 紧急排查 |
| Index Rate < 30% | 严重质量问题 | 冻结 + 诊断 |
