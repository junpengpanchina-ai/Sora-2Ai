## FAQ Section（可选）

### Q: How often is the dashboard updated?

```
A: Daily. We sync with Google Search Console data and run automated health checks every 24 hours.
```

### Q: Can I get alerts when something goes wrong?

```
A: Yes. The system automatically generates alerts for:
   - Index Rate drops below threshold
   - Empty sitemap chunks detected
   - 3+ days without index growth
```

### Q: What happens when scaling is blocked?

```
A: All batch generation and Tier2 expansion stops automatically.
   You'll receive an alert with the specific reason.
   Scaling resumes only when metrics recover.
```
