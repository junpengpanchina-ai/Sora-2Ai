# Index Health Dashboard（索引健康仪表盘）

> **目标**：用 5 个数判断当前状态，不用 site:，不用猜，不用焦虑  
> **适用对象**：新站 / programmatic / GEO 内容，10k–300k URL 规模

---

## 🎯 核心原则

**Google 吞不下 15 万页面，不是因为不喜欢你，而是因为你没让它"安心"。**

你现在这套 GEO-A v2 + sitemap + 节奏，已经在让 Google 安心了。

---

## 📊 5 个核心指标（全部来自 GSC）

| 编号 | 指标 | GSC 路径 | 说明 |
|------|------|----------|------|
| **①** | Discovered – not indexed | 索引 → 网页 | Google 发现了但还没索引 |
| **②** | Crawled – not indexed | 索引 → 网页 | Google 抓取了但还没索引 |
| **③** | Indexed | 索引 → 网页 | 已被索引的页面数 |
| **④** | Crawl requests/day | 设置 → 抓取统计 | 每日抓取请求数 |
| **⑤** | Average response time | 设置 → 抓取统计 | 平均响应时间 |

---

## 🧠 核心判断逻辑（一眼看状态）

### 🟢 状态一：慢吃期（最理想）

**👉 Google 在"排队消化你"**

#### 指标特征

| 指标 | 正常范围 |
|------|----------|
| ① Discovered – not indexed | **持续上升** |
| ② Crawled – not indexed | **缓慢上升或持平** |
| ③ Indexed | **每天小幅增长** |
| ④ Crawl/day | **稳定**（不需要高） |
| ⑤ Response time | **< 800ms** |

#### 解读

- ✅ Google 信任你
- ✅ 但还在评估 + 控速
- ✅ 这是 programmatic 的黄金阶段

#### ✅ 行动表（只能做这些）

**✔ 必须做**：
- 维持当前发布节奏（20–40 / 天）
- 不改模板
- 不加热点
- 持续监控指标

**❌ 绝对不做**：
- 不提速
- 不加 sitemap
- 不改内链结构
- 不追热点词

**👉 你现在 = 99% 在这个阶段**

---

### 🟡 状态二：限速期（可控，但要克制）

**👉 Google 觉得你"有点快了"**

#### 指标特征

| 指标 | 表现 |
|------|------|
| ① Discovered | **继续涨** |
| ② Crawled | **明显堆积**（Crawled 数量远大于 Indexed） |
| ③ Indexed | **停或涨很慢** |
| ④ Crawl/day | **不涨甚至下降** |
| ⑤ Response time | **正常**（< 800ms） |

#### 解读

- ⚠️ Google 没有否定你
- ⚠️ 但在说：「我先别吃那么多」
- ⚠️ 这是"踩刹车"，不是惩罚

#### ✅ 行动表

**🔻 立即行动**：
- **发布量减半**（从 20-40 / 天 → 10-20 / 天）
- **冻结模板 ≥ 14 天**（不要任何修改）
- **强化 hub → longtail 内链**（提升核心页权重）

**❌ 绝对不做**：
- 不要任何新"策略尝试"
- 不要提速
- 不要改模板结构
- 不要加新功能

**监控频率**：每 3 天检查一次，直到 Crawled 堆积缓解

---

### 🔴 状态三：风险期（必须止血）

**👉 Google 开始怀疑"你是不是在刷内容"**

#### 指标特征

| 指标 | 表现 |
|------|------|
| ① Discovered | **不涨**（甚至下降） |
| ② Crawled | **不涨或下降** |
| ③ Indexed | **停滞甚至回落** |
| ④ Crawl/day | **明显下降** |
| ⑤ Response time | **正常**（< 800ms） |

#### 解读

- 🚨 不是算法惩罚
- 🚨 是信任度下降
- 🚨 必须立即止血

#### ✅ 行动表（必须）

**⛔ 立即行动**：
- **立刻停止发布 7–14 天**
- **只保留 core / use-case 层**（删除低质量页面）
- **停掉最相似的 10–20% 页面**（降低同构风险）

**❌ 绝对不做**：
- 绝对不碰热点词
- 绝对不改模板结构
- 不要尝试新策略
- 不要加速发布

**恢复策略**：
- 14 天后重新评估
- 如果 Discovered 开始回升，可以恢复发布（10-20 / 天）
- 如果仍然不涨，再等 14 天

---

## 🧩 黄金判断公式（记住这一条就够）

```
Discovered 在涨 = Google 在"看你"
Crawled 在涨 = Google 在"试你"
Indexed 在涨 = Google 在"认你"
```

**三者顺序一定是**：
```
Discovered → Crawled → Indexed
```

**如果你想跳过前两个，👉 Google 会直接不认你。**

---

## 🔍 3 分钟自检法

### 每天 / 每 3 天做一次

1. **打开 GSC** → 索引 → 网页
2. **记录 3 个数**：
   - Discovered – not indexed
   - Crawled – not indexed
   - Indexed
3. **对比 7 天前**的数据
4. **判断**：
   - ✅ **只要 Discovered 在涨** → 一切 OK（慢吃期）
   - ⚠️ **Crawled 堆积**（Crawled 远大于 Indexed） → 减速（限速期）
   - 🚨 **Indexed 回落** → 停（风险期）

### 快速判断表

| Discovered | Crawled | Indexed | 状态 | 行动 |
|------------|---------|---------|------|------|
| ↑ 上升 | ↑ 缓慢上升 | ↑ 小幅增长 | 🟢 慢吃期 | 维持节奏 |
| ↑ 上升 | ↑↑ 明显堆积 | → 停滞 | 🟡 限速期 | 发布量减半 |
| → 不涨 | → 不涨/下降 | ↓ 回落 | 🔴 风险期 | 停止发布 |

## 📋 详细指标阈值

### 慢吃期（🟢）

| 指标 | 阈值 | 说明 |
|------|------|------|
| Discovered 增长率 | > 0（持续上升） | 每天都有新页面被发现 |
| Crawled / Indexed 比例 | < 3:1 | Crawled 不会过度堆积 |
| Indexed 增长率 | > 0（小幅增长） | 每天都有新页面被索引 |
| Crawl/day | 稳定（不需要高） | 不需要追求高抓取量 |
| Response time | < 800ms | 服务器响应正常 |

### 限速期（🟡）

| 指标 | 阈值 | 说明 |
|------|------|------|
| Discovered 增长率 | > 0（继续涨） | Google 还在发现新页面 |
| Crawled / Indexed 比例 | > 3:1 | Crawled 明显堆积 |
| Indexed 增长率 | ≈ 0（停滞） | 索引速度明显下降 |
| Crawl/day | 不涨或下降 | Google 在控制抓取 |
| Response time | < 800ms | 服务器响应正常 |

### 风险期（🔴）

| 指标 | 阈值 | 说明 |
|------|------|------|
| Discovered 增长率 | ≤ 0（不涨或下降） | Google 停止发现新页面 |
| Crawled / Indexed 比例 | 下降 | Crawled 也在下降 |
| Indexed 增长率 | < 0（回落） | 已索引页面数下降 |
| Crawl/day | 明显下降 | Google 大幅减少抓取 |
| Response time | < 800ms | 服务器响应正常（排除技术问题） |

## 📊 监控频率建议

| 阶段 | 监控频率 | 检查指标 |
|------|----------|----------|
| **慢吃期** | 每 3 天 | ① Discovered ② Crawled ③ Indexed |
| **限速期** | 每 3 天 | ① Discovered ② Crawled ③ Indexed ④ Crawl/day |
| **风险期** | 每天 | 全部 5 个指标 |

1. Discovered – not indexed
2. Crawled – not indexed
3. Indexed
4. Crawl requests/day
5. Average response time
